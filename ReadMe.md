# 厦门文化展示网站

这是一个展示厦门(Amoy)特色文化的现代化网站项目，通过多媒体形式全方位展现厦门的人文风貌、美食文化、自然景观和闽南语言特色。项目现已全面采用 Vue.js 重构，提供更好的用户体验和开发维护性。

## 项目特点

- **现代化框架**：基于 Vue.js 3.x 构建，数据驱动的响应式网站
- **多维度展示**：厦门文化，包括景点、美食、文化习俗和方言
- **丰富多媒体**：图片、音频和视频资源，支持交互式体验
- **响应式设计**：提供良好的移动端和桌面端浏览体验
- **组件化架构**：模块化设计，易于维护和扩展
- **交互增强**：音频播放控制、平滑滚动导航等现代化交互

## 功能模块

### 1. **美景展示** (`meijing_vue.html`)

- 🏞️ **4个主要景点**：鼓浪屿、白城沙滩、厦门大学、沙坡尾
- 📱 **动态数据管理**：Vue 数据驱动的景点信息展示
- 🖼️ **响应式图片展示**：高质量图片展示与描述
- 🧭 **平滑滚动导航**：点击跳转到指定景点介绍
- 💡 **计算属性增强**：动态计算和数据处理

### 2. **美食文化** (`meishi_vue.html`)

- 🍜 **3种特色美食**：沙茶面、土笋冻、姜母鸭
- 📊 **数据结构化管理**：美食信息的系统化组织
- ⭐ **动态评分系统**：预留用户评价功能
- 🔍 **美食搜索功能**：支持关键词搜索（预留）
- 🏷️ **标签分类系统**：美食类型和特色标签

### 3. **文化习俗** (`wenhua_vue.html`)

- 🎌 **4大文化主题**：闽南语、市花市树、厦门地铁、博饼
- 🎵 **闽南语音频播放系统**：支持 11 个音频文件交互播放
- 🎲 **博饼规则动态展示**：传统中秋文化互动介绍
- 🚇 **地铁特色组件化**：厦门地铁文化专题展示
- 🎭 **文化主题管理**：动态文化内容组织

### 4. **方言特色**

- 🗣️ **闽南语音频教学**：8个常用词汇 + 3个句子
- 📚 **常用词汇展示**：什么、加油、真棒、多谢等
- 🎮 **互动学习功能**：点击播放音频，支持音频切换控制
- 🎵 **音乐推荐**：《爱拼才会赢》等经典闽南语歌曲

## 本地运行

### 🚀 **快速开始**

1. 克隆项目到本地

   ```bash
   git clone https://github.com/brilliant751/Amoy.git
   cd Amoy
   ```

2. **直接运行**（推荐Vue版本）

   ```bash
   # 使用浏览器打开 Vue 版本页面
   file/meijing.html
   file/meishi.html
   file/wenhua.html
   ```
